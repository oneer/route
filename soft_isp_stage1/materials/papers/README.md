## 阅读要求

每篇论文只产出一页笔记，使用 `materials/notes/paper_reading_template.md`。不要陷入全文精读，先回答：

1. 它处理什么输入数据域？
2. 输出是什么数据域？
3. 对 Soft-ISP 阶段 1 哪个模块有帮助？
4. 哪些内容暂时不要深挖？
