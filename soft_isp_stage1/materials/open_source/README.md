# 开源项目阅读清单

## 只读结构，不照搬代码

| 项目 | 链接 | 读什么 |
|---|---|---|
| OpenISP | https://github.com/cruxopen/openISP | 模块拆分、配置组织、pipeline 调用顺序。 |
| Infinite-ISP | https://github.com/10x-Engineers/Infinite-ISP | 完整 ISP 模块命名、输入输出、文档组织。 |

## 阅读顺序

1. 先自己实现最小版本。
2. 再看开源项目如何命名和组织。
3. 只把“结构和验证思路”记到笔记，不复制实现。

## 输出

在 `materials/notes/open_source_structure_notes.md` 写：

- 这个项目有哪些模块？
- 和你的 `soft_isp/` 模块差异是什么？
- 哪些命名或配置方式值得借鉴？
