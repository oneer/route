# Lecture 17：机器人学习 / World Modeling

## 这讲在讲什么

不同年份课程中 Lecture 17 可能叫机器人学习或 world modeling。核心都是让模型理解环境、动作和未来状态。

## 核心概念

| 概念 | 解释 | 和 AI-ISP 的关系 |
|---|---|---|
| Robot learning | 让机器人从数据中学习感知和控制 | 非当前主线 |
| World model | 学习环境状态转移 | 和视频预测相关 |
| Policy | 根据观测决定动作 | 控制方向 |
| Embodied AI | 视觉、动作、环境闭环 | 下游应用 |

## 为什么当前先跳过

机器人学习是感知 + 决策 + 控制。AI-ISP 当前是图像恢复：

```text
bad image -> better image
```

两者层级不同。ISP 是机器人/自动驾驶/AR 等系统的输入前端之一，但不是控制策略本身。

## 间接关系

低光、运动模糊、噪声会影响机器人感知。如果 ISP 做得不好，下游定位、检测、避障都可能变差。

但你当前先要把图像质量本身做好。

## 最小带走

1. AI-ISP 可以服务具身智能，但不是同一个学习阶段。
2. 当前不需要学机器人控制。
3. 后续如果做视频/实时相机系统，再关注世界模型和时序感知。
